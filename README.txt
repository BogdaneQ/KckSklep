Sklep Sportowy - Aplikacja Konsolowa
Opis
"Sklep Sportowy" to aplikacja konsolowa, która symuluje funkcjonalności sklepu internetowego w środowisku tekstowym. Użytkownik może przeglądać produkty, zarządzać koszykiem. Aplikacja została zaprojektowana w celu nauki i demonstracji operacji na produktach oraz interfejsów konsolowych.


Mechanika działania

Produkty:
Każdy produkt posiada unikalne ID, nazwę, opis, cenę oraz liczbę dostępnych sztuk.
Produkty są organizowane w kategorie, co ułatwia ich przeglądanie.

Koszyk:
Przechowuje listę wybranych produktów wraz z ilością.
Koszyk działa w trybie sesji, co oznacza, że jego zawartość jest usuwana po zakończeniu działania aplikacji.

Interfejs użytkownika:
Aplikacja wykorzystuje kolorowe wyświetlanie tekstu do podkreślania kluczowych elementów, np. aktualnie wybranej opcji lub błędów.
Wszystkie dane wejściowe od użytkownika są walidowane, aby uniknąć błędów (np. wymagane liczby przy podawaniu ilości produktów lub cen).

Instrukcja użytkowania

Uruchom aplikację.
Przejdź do przeglądania produktów, dodawania ich do koszyka lub przeglądania koszyka.
Finalizuj zamówienie, podając adres dostawy.
W dowolnym momencie możesz wyjść z aplikacji.

Wymagania
System operacyjny z obsługą konsoli tekstowej (np. Windows, Linux, macOS).
.NET Framework lub .NET Core do uruchomienia aplikacji.
